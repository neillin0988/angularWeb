ng-show / ng-hide : 是將元件 display: none !important; or not

[this is divA ng-show="valueA"] SHOW when [valueA] is true
[this is divB ng-hide="valueA"] HIDE when [valueA] is true

ng-if : when false, HTML code 會消失

[this is divA ng-if="valueA"] SHOW when [valueA] is true
[this is divB ng-if="valueA"] HIDE when [valueA] is true
